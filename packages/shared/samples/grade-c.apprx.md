## 측정 인프라 세팅 필요
지표가 없으면 진단도 처방도 추측입니다.
측정 체계를 갖추는 것이 첫 번째 처방입니다.

### Day 1~3: 측정 인프라 설치
- [ ] Firebase 프로젝트 생성 + Analytics 세팅 (30분~1시간)
  → google-services.json 추가, SDK 초기화
  → 핵심: 별도 이벤트 코드 없이도 기본 지표(DAU, MAU, 세션) 자동 수집
- [ ] AdMob 계정 생성 + 배너 광고 1개 삽입 (1시간)
  → 수익화 첫 발걸음, 복잡한 전략 전에 "돈이 들어오는 경험"부터

### Day 4~10: 데이터 축적 기간
- 매일 3분 체크 루틴:
  → Firebase Console > Analytics > 실시간 확인
  → 확인할 것: 오늘 활성 사용자 수, 세션 수
- 이 기간에 코딩할 것은 없음. 데이터가 쌓이길 기다리는 시간.

### Day 11~14: 첫 데이터 리뷰

- → 본 진단 (수익화 단계, 벤치마크 비교, 시뮬레이션) 자동 활성화
